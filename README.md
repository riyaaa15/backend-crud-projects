# Backend CRUD Projects
This repository contains **CRUD**  projects built while practicing Node.js, Express, and database integration with MySQL and MongoDB.

## Projects
1. mysql-user-manager - A full CRUD user management app built with Node.js, Express, MySql2, and Ejs. Support adding editing username with password verification, and deleting users with email and password confirmation. Uses Faker.js for bulk dummy data genertion and dotenv for secure DB credentials.

2. whatsapp-chat-crud - A WhatsApp-style chat CRUD app built with Node.js, Express, MongoDB, Mongoose, and EJS. Supports creating, viewing, editing, and deleting chat messages with sender and receiver fields. Uses method-override for PUT and DELETE from HTML forms.
