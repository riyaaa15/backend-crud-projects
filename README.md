# Backend CRUD Projects
This repository contains **CRUD**  projects built while practicing Node.js, Express, and database integration with MySQL and MongoDB.

## Projects
1. mysql-user-manager - A full CRUD user management app built with Node.js, Express, MySql2, and Ejs. Suppoer adding editing username with password verification, and deleting users with email and password confirmation. Uses Faker.js for bulk dummy data genertion and dotenv for secure DB credentials.
